# Spatial Transcriptomics Analysis (Cyberomix-style Implementation)

空間トランスクリプトームデータ（Visium HD）に対して、
核セグメンテーションと遺伝子発現情報を統合し、
細胞単位での空間遺伝子発現解析を行った実装例です。  
（守秘義務に基づき、公開チュートリアルデータを用いて再構成した解析例を掲載）

本解析では、H&E画像の位置情報と遺伝子発現データを統合し、
核単位での遺伝子発現マッピングおよびクラスタリング解析までを
Pythonで一貫して実装しています。

---

## 解析フロー

- H&E画像の前処理・正規化（percentile normalization）
- StarDistによる核セグメンテーション（nuclei detection）
- セグメンテーション結果のGeoDataFrame化（空間ポリゴン生成）
- Visium HD遺伝子発現データの読み込み（AnnData）
- 空間座標と遺伝子発現の統合（spatial join）
- 核単位での遺伝子カウント集約（custom binning）
- QC（UMI分布・核面積フィルタリング）
- PCA・近傍グラフ構築・Leidenクラスタリング
- 遺伝子発現およびクラスタの空間可視化

---

## 解析結果例

- 核セグメンテーションに基づく細胞単位の空間解析
- 特定遺伝子（例：Lyz1, Muc2等）の空間発現マッピング
- Leidenクラスタリングによる空間的細胞集団の可視化
- UMI分布・核面積に基づく品質フィルタリング

（※実データに依存するため、代表的な解析フローの再構成例を掲載）

## 使用技術

Python / Scanpy / Squidpy / StarDist / GeoPandas  
NumPy / pandas / matplotlib / AnnData  
空間トランスクリプトーム解析・画像解析・空間統計処理を統合した実装

---

## 実装内容

・核セグメンテーション（StarDist）による細胞領域抽出  
・画像座標と遺伝子発現データの空間統合処理（spatial join）  
・核単位での遺伝子発現カウントの再集約（custom binning実装）  
・QC指標（核面積・UMI数）に基づくフィルタリング設計  
・PCA・近傍グラフ・Leidenクラスタリングによる細胞集団解析  
・遺伝子発現およびクラスタの空間可視化パイプライン構築  

特に、画像由来の位置情報と遺伝子発現データを
前処理から解析・可視化まで一体として設計し、
再現性と運用性を意識した解析フローとして実装した点に特徴があります。
